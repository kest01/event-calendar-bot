Проект календаря событий для сообщества в Telegram. Может использоваться для ведения календаря совместных мероприятий, встреч, забегов и т.д. Участиники сообщества могут создавать и редактировать мероприятия, отмечаться о статусе своего участия, просматривать кто еще будет участвовать.

Проект реализуется как Telegram Mini App. 
 - В папке frontend фронтовая часть приложения на базе vite. 
 - В папке backend бэковая часть Telegram бота, а также рест сервисы, к которым обращается фронт.
